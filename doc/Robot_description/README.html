Concepts to write the description of a robot

  • URDF
  • XACRO
  • DH description

Robot design can be done in 3d modeling environment. SOLIDWORKS seems to be the most common for professional use cases. You may be able to use Blender or other free/open source alternatives.

Note

SOLIDWORKS provides tools to automatically get the URDF of a model. However, the model should be constructed in a precise way in order to obtain the right link and joint information. Instead, it might be easier to write an URDF with simple shapes.
